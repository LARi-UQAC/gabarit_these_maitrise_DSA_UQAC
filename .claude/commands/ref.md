# Academic reference formatting and validation

Format, verify, or find the following academic reference per LAR.i / UQAC requirements.

**Accepted publishers only**: IEEE, Springer, Elsevier, Taylor & Francis, Cambridge, Wiley, IET, IOP, ACME, MDPI.
Any other publisher must be submitted to the professor for approval before inclusion.

**Required LaTeX format**:

- DOI mandatory, hyperlinked via `\href{http://doi.org/...}{DOI}` in the reference list
- Language: English or the publication's original language
- The reference must genuinely exist — do not invent or infer a DOI

**For each reference, provide**:

1. Full reference formatted in LaTeX (BibTeX or natbib style as appropriate)
2. Clickable DOI
3. LaTeX comment with confidence level (0–100 %) between the paper's content and the citing text, plus a one-sentence justification
4. A minimum introductory sentence presenting the reference in the text

If the reference cannot be found or the DOI cannot be verified, state this explicitly — do not fabricate a reference.
